API Documentation

A movie directory containing nature movies or movies that feature beautiful landscapes and nature scenes.

You can search for movies by title, director, or year. New users can register and login to the site. Users can add movies to their watchlist.

Base-Url: http://127.0.0.1:8080/

Response content type: application/json

/movies
/users

Movie Endpoints

Returns a list of all movies in the database.

Response:

                
                   [
                      {
                        "Title": "string",
                        "Description": "string",
                        "Director":  {
                          "Name": "string",
                          "Description": "string"
                        },
                        "Year": number,
                        "Featured": boolean,
                      } 
                    ]

Returns a single movie by id.

Response:

                
                  {
                    "Title": "string",
                    "Description": "string",
                    "Director":  {
                      "Name": "string",
                      "Description": "string"
                    },
                    "Year": number,
                    "Featured": boolean,
                  }

Returns data about a movie genre by name.

Response:

                
                  {
                    "Genre":  {
                      "Description": "string"
                    },
                  }

Returns information about a director.

Response:

                
                  {
                    "Director":  {
                      "Description": "string",
                      "Name": "string"
                    },
                  }

User Endpoints

Creates a new user.

Body:*required

                
                  {
                    "Username": "string",
                    "Password": "string",
                    "Email": "string",
                    "Birthday": Date,
                  }

Updates a user's information.

Body:*required

                
                  {
                    "Username": "string",
                    "Password": "string",
                    "Email": "string",
                    "Birthday": Date,
                  }

Adds a movie to the user's watchlist.

Response:

                
                  {
                    "Username": "string",
                    "Password": "string",
                    "Email": "string",
                    "FavoriteMovies": []
                  }

Removes a movie from the user's watchlist.

Response:

                
                  {
                    "Username": "string",
                    "Password": "string",
                    "Email": "string",
                  }

Removes & deregisters a user

Response:

                
                  ${Username} was deleted.
